Boeing 737-800 Interactive Checklist (APU Only)

A detailed guide for a full flight from a cold & dark state to shutdown, including FMC steps and interactive checklists. Intended for flight simulation use only.

Phase 1: Pre-Flight & Cockpit Preparation

Detailed Pilot Actions (Switches, Buttons & FMC Steps) Avionics & Autopilot Display ATC Communication
  • Set BATTERY master switch to ON and close guard.
  • Set STANDBY POWER switch to AUTO.
  • On Fuel Panel, set left AFT FUEL PUMP to ON.
  • On Electrical Panel, hold APU START switch to START (2 sec), then release to ON.
  • Set both APU GENERATOR switches to ON after "APU GEN OFF BUS" light illuminates.
  • Set both IRS Mode Selectors from OFF to NAV.
  • Set both ELECTRIC HYDRAULIC PUMP switches to ON.
  • Set all four WINDOW HEAT switches to ON.
  • Set APU BLEED air switch to ON.
  • Set both PACK switches to AUTO.
  • Set ISOLATION VALVE switch to AUTO.
  • Set LANDING ALTITUDE on Pressurization Panel.
  • On Glareshield, turn both FLIGHT DIRECTOR (F/D) switches ON.
  • On MCP, set initial cleared altitude in ALTITUDE window.
  • On CDU, access FMC and press IDENT to verify aircraft data.
  • On CDU, press POS INIT, confirm GPS position and set initial position.
  • On CDU, press RTE, enter Origin, Destination, and route waypoints. Activate and Execute.
  • On CDU, press PERF INIT, enter Cost Index, Cruise Alt, Reserves, and ZFW. Execute.
  • On CDU, press N1 LIMIT, select takeoff thrust rating (e.g., TO-1, TO-2) or leave full.
  • On CDU, press TAKEOFF, enter Flap setting, verify V-speeds, and input Trim.
  • On radio panel, tune to Clearance Delivery frequency.
 Initial: Standby instruments and FMC screen are on.

After APU Power: All DUs illuminate. EICAS indicates APU GENs are powering the aircraft. ND shows "IRS ALIGNING".

After FMC/MCP Setup: PFD shows green "FD" and command bars. V-speeds appear on speed tape. ND shows magenta flight plan route. 		Pilot: "Clearance Delivery, [Aircraft Callsign], stand [Stand Number], request IFR clearance to [Destination Airport] with information [ATIS Code]."

ATC: "[Callsign], cleared to [Destination] via [Route/SID], climb and maintain [Initial Altitude], squawk [Transponder Code]."

Pilot (Read back): "Cleared to [Destination] via [SID], climb [Altitude], squawk [Code], [Callsign]."

Phase 2: Pushback & Engine Start

Detailed Pilot Actions (Switches, Buttons & FMC Steps) Avionics & Autopilot Display ATC Communication
  • Set ANTI-COLLISION (BEACON) light switch to ON.
  • On MCP, press LNAV and VNAV buttons.
  • Set both PACK switches to OFF.
  • Turn ENGINE 2 START SWITCH to GRD.
  • At ~25% N2, move ENGINE 2 START LEVER to IDLE.
  • After Engine 2 stabilizes, turn ENGINE 1 START SWITCH to GRD.
  • At ~25% N2, move ENGINE 1 START LEVER to IDLE.
  • Set both ENGINE GENERATOR switches to ON.
  • Set both PACK switches to AUTO.
  • Set APU BLEED switch to OFF.
  • Set APU switch to OFF.
  • Set PROBE HEAT switches to ON.
  • Set ENGINE START SWITCHES back to AUTO.
 FMA: LNAV and VNAV appear in white (armed).

EICAS: "START VALVE OPEN" is shown during start. N2, EGT, FF, N1, and Oil Pressure rise sequentially. After start, EICAS shows engine generators are powering the buses. 		Pilot: "Ground, [Callsign], at stand [Number], request pushback and engine start."

ATC: "[Callsign], pushback and start approved, face [Direction], expect runway [Runway]."

Pilot: "Pushback start approved, face [Direction], [Callsign]."

Phase 3: Taxi

Detailed Pilot Actions (Switches, Buttons & FMC Steps) Avionics & Autopilot Display ATC Communication
  • Move FLAP LEVER to takeoff setting (e.g., 5).
  • Check flight controls for full and free movement.
  • Set TAXI LIGHT switch to ON.
  • Set AUTOBRAKE selector to RTO (Rejected Takeoff).
 PFD/EICAS: Flap indicator moves to the selected setting.

EICAS: "AUTOBRAKE RTO" is displayed. 		Pilot: "Ground, [Callsign], request taxi."

ATC: "[Callsign], taxi to runway [Runway] via taxiway [Taxiway Route]."

Pilot (Read back): "Taxi to runway [Runway] via [Route], [Callsign]."

Phase 4: Takeoff and Climb

Detailed Pilot Actions (Switches, Buttons & FMC Steps) Avionics & Autopilot Display ATC Communication
  • Set LANDING LIGHTS switches to ON.
  • Set STROBE LIGHT switch to ON.
  • On MCP, set the A/T ARM (Autothrottle) switch to ARM.
  • Advance thrust levers to ~40% N1.
  • Press the TO/GA (Takeoff/Go-Around) button on the thrust levers.
  • At "Rotate" call, pull back on control column, following FD bars.
  • With positive climb rate, move LANDING GEAR LEVER UP.
  • At safe altitude (~1000 ft AGL), press the CMD A or CMD B button on the MCP.
 PFD FMA: "A/T ARM" (white) -> "N1" (green) for thrust, "TO/GA" (green) for pitch. After AP engagement, "CMD" appears in green, and roll/pitch modes become "LNAV" and "VNAV SPD".

EICAS: Gear indication changes from green to in-transit, then disappears. 		Pilot: "Tower, [Callsign], ready for departure runway [Runway]."

ATC: "[Callsign], wind [Direction/Speed], runway [Runway], cleared for takeoff."

Pilot: "Cleared for takeoff, runway [Runway], [Callsign]."

ATC: "[Callsign], contact Departure on [Frequency]."

Pilot: "Contacting Departure, [Callsign]."

Phase 5: Cruise & Descent

Detailed Pilot Actions (Switches, Buttons & FMC Steps) Avionics & Autopilot Display ATC Communication
  • Monitor as Autopilot levels off at cruise altitude.
  • Move LANDING GEAR LEVER to the OFF position.
  • Turn off TAXI and TURNOFF lights.
  • On MCP, set new lower altitude from ATC for descent.
  • On NAV radio panel, tune the ILS FREQUENCY for the approach runway.
  • Press the APP (Approach) button on the MCP.
  • Move FLAP LEVER and LANDING GEAR LEVER down on schedule.
 PFD FMA (Cruise): Pitch mode becomes "VNAV PTH" or "ALT".

PFD FMA (Descent/Approach): "VNAV PTH" manages descent. For an ILS, "VOR/LOC" and "G/S" appear in white (armed), then turn green (captured).

EICAS: Three green lights confirm gear is down and locked. 		ATC: "[Callsign], contact [Center/Control] on [Frequency]."

Pilot: "[Center/Control], [Callsign], flight level [Cruise Level]."

ATC: "[Callsign], descend via the [STAR Name] arrival."

Pilot: "Descend via the [STAR Name], [Callsign]."

ATC: "[Callsign], cleared ILS runway [Runway] approach."

Pilot: "Cleared ILS runway [Runway] approach, [Callsign]."

Phase 6: Landing and Shutdown

Detailed Pilot Actions (Switches, Buttons & FMC Steps) Avionics & Autopilot Display ATC Communication
  • At decision height, press AUTOPILOT DISCONNECT SWITCH on yoke.
  • After touchdown, deploy SPEEDBRAKE LEVER.
  • Engage REVERSE THRUST LEVERS as needed.
  • After clearing the runway, set STROBE LIGHT to STEADY.
  • Set LANDING LIGHTS and PROBE HEAT to OFF.
  • Move FLAP LEVER to UP.
  • Set the APU START switch to ON.
  • At the gate, set PARKING BRAKE.
  • Set APU GENERATOR switches to ON.
  • Move both ENGINE START LEVERS to CUTOFF.
  • Set ANTI-COLLISION (BEACON) light OFF.
  • Turn off FUEL PUMPS, HYDRAULIC PUMPS, PACKS, WINDOW HEAT.
  • Set APU switch to OFF.
  • Set both IRS Mode Selectors to OFF.
  • Set the BATTERY master switch to OFF.
 PFD FMA: "CMD" disappears upon AP disconnect. FD bars remain for guidance.

EICAS: Shows speedbrake and reverse thrust indications. APU start sequence begins after landing. At shutdown, EICAS shows power transfer to APU, then engine parameters spool down. Finally, all displays go dark. 		ATC: "[Callsign], wind [Wind], runway [Runway], cleared to land."

Pilot: "Cleared to land, [Callsign]."

ATC: "Contact Ground on [Frequency]."

Pilot: "Ground, [Callsign], clear of runway [Runway] at [Taxiway], taxi to the gate."

ATC: "[Callsign], taxi to gate [Number] via [Route]."
